 # AI 量化回测方法论检查清单
 
 > 格式说明：每项以 `- [ ]` 开头，Codex 可直接解析。  
 > `✅` = 通过, `❌` = 未通过, `N/A` = 不适用。
 
 ## 一、方法论文档化（8 项方法论缺陷检查）
 
 ### 1.1 数据划分
 - [ ] 训练/验证/测试集的日期边界是否在代码中有明确注释？
 - [ ] 测试集是否只跑过 1 次（未回头调参）？
 - [ ] 策略选择是否在验证集上做的（不在测试集上）？
 - [ ] 样本外测试的独立调仓次数是否 ≥ 12？
 - [ ] 参数搜索范围是否记录在日志中？
 
 ### 1.2 数据清洗
 - [ ] 异常值是否已 Winsorize（±5σ）？
 - [ ] 停牌日是否已前向填充（≤3 天）？
 - [ ] 复权是否已进行连续性校验（相邻日涨跌幅不超过 ±20%）？
 - [ ] 是否存在因停牌超 3 天而被 drop 的数据段记录？
 - [ ] 数据清洗日志是否输出了异常值/停牌/填充的统计量？
 
 ### 1.3 模型评估
 - [ ] HMM 的 BIC/AIC 是否记录？
 - [ ] 情绪模型的准确率（Accuracy）是否评估？
 - [ ] 滚动窗口参数（vol, corr）是否基于扩窗历史计算？
 - [ ] 回测结果是否包含了完整的评估指标（≥18 项）？
 
 ### 1.4 过拟合防范
 - [ ] 是否计算了 Deflated Sharpe Ratio？
 - [ ] 是否计算了 PBO（Probability of Backtest Overfitting）？
 - [ ] 参数敏感性分析是否完成（至少 ±20%）？
 - [ ] 最佳策略 vs 次好策略的差距是否合理（差距过大 = 过拟合）？
 - [ ] 是否对比了等权 ETF 组合（不调仓）作为基线？
 
 ### 1.5 成本真实性
 - [ ] 佣金是否 ≥ 万 2.5？
 - [ ] 最低佣金（5 元）是否建模？
 - [ ] 滑点是否 ≥ 2bp？
 - [ ] 冲击成本是否随参与率非线性增长？
 - [ ] 年化换手率是否 ≤ 300%？
 - [ ] 无券商手续费时是否会 warn（不阻断）？
 
 ### 1.6 元学习
 - [ ] Meta-Learner 是否记录 ≥ 8 个季度的历史？
 - [ ] Builder/Critic 的信誉分是否在线更新？
 - [ ] 被 REJECT 的提案权重是否自动切换到防守策略？
 
 ### 1.7 端到端验证
 - [ ] 回测日志是否记录了每次调仓的 train_end, test_start, test_end？
 - [ ] 策略池是否可注入（不硬编码策略列表）？
 - [ ] 结果是否包含净值曲线 + 每步权重 + 状态标签？
 - [ ] 全程不报错时，Critic 是否至少触发过 1 次 DOWNGRADE 或 REJECT？
 
 ## 二、7 Deadly Sins 检查（Lopez de Prado）
 
 ### 2.1 Survivorship Bias（幸存者偏差）
 - [ ] 回测标的包含目前已退市/合并的 ETF 吗？（若否，需记录说明）
 - [ ] 数据获取是否使用历史成分股/ETF 列表而非当前快照？
 
 ### 2.2 Look-Ahead Bias（前视偏差）
 - [ ] HMM 是否只在训练集上 fit？（检查是否有全局标准化）
 - [ ] 是否设置了 Purging gap（≥5 交易日）？
 - [ ] 是否设置了 Embargo gap（≥10 自然日）？
 - [ ] 滚动指标（vol, corr）是否基于扩窗历史计算而非全量数据？
 
 ### 2.3 Data Snooping（数据窥探）
 - [ ] 测试集只跑了 1 次（未回头调参）？
 - [ ] 参数搜索是否在验证集上完成（不在测试集上）？
 - [ ] PBO 是否 < 0.3？
 - [ ] Deflated Sharpe 对应的 p-value 是否 < 0.05？
 
 ### 2.4 Ignoring Transaction Costs（忽略交易成本）
 - [ ] 佣金是否 ≥ 万 2.5？
 - [ ] 最低佣金（5 元）是否建模？
 - [ ] 滑点是否 ≥ 2bp？
 - [ ] 冲击成本是否随参与率非线性增长？
 - [ ] 年化换手率是否 ≤ 300%？
 
 ### 2.5 Outlier-Driven Success（极端值驱动）
 - [ ] 净值曲线是否在去掉极端日（±5σ 以外）后仍为正？
 - [ ] 月度收益贡献分布是否均匀？
 - [ ] 最大回撤是否由少数极端日引发？
 
 ### 2.6 Shorting Constraints（做空约束）
 - [ ] 所有权重是否 ≥ 0？
 - [ ] 权重和是否 = 1（满仓）？
 - [ ] 单标的权重是否 ≤ 40%？
 
 ### 2.7 Storytelling（事后叙事）
 - [ ] 策略逻辑是否有明确的数学/统计依据？
 - [ ] 是否对比了沪深 300 买入持有？
 - [ ] 是否对比了等权 ETF 组合（不调仓）？
 - [ ] 是否分牛/熊/震荡状态分别报告？
 - [ ] 是否计算了扣除时间成本后的净超额收益？
 
 ## 三、全局禁止项检查
 - [ ] 是否在全部数据上 fit 任何模型再"回测"？
 - [ ] 是否在测试集上跑完后回头调参？
 - [ ] 是否用全局 mean/std 做标准化？
 - [ ] 是否假设能以收盘价成交？
 - [ ] 是否忽略最低佣金？
 - [ ] 是否只报告 Sharpe？（必须包含 DSR 和 PBO）
 - [ ] 报告是否诚实（不编造好看的结果）？
